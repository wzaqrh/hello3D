    IDirect3DDevice9::SetPixelShaderConstantB method

    • 2018/12/05
    • 2 分钟阅读时长

    本文内容

    1. Syntax
    2. Parameters
    3. Return Value
    4. Requirements
    5. See Also

    Sets a Boolean shader constant.

    Syntax

    C++ 
    HRESULT SetPixelShaderConstantB(
  UINT       StartRegister,
  const BOOL *pConstantData,
  UINT       BoolCount
);

    Parameters

    StartRegister

    Type: UINT

    Register number that will contain the first constant value.

    pConstantData

    Type: const BOOL*

    Pointer to an array of constants.

    BoolCount

    Type: UINT

    Number of boolean values in the array of constants.

    Return Value

    Type: HRESULT

    If the method succeeds, the return value is D3D_OK. If the method fails, the return value can be D3DERR_INVALIDCALL.

    Requirements

       
    Target Platform Windows
    Header d3d9.h (include D3D9.h)
    Library D3D9.lib
